## Git

Usefull github commands:

1. ```git init```
2. ```git clone <github_url>```
3. ```git remote add origin <github_url>```
4. ```git push```
5. ```git pull```
6. ```git commit```
7. ```git commit -m <message_for_your_commit>```
8. ```git pull --rebase```
9. ```git stash```
10. ```git stash pop```
11. ```git reset --hard <commit_id>```
12. ```git commit --amend```
13. ```git status```
14. ```git diff```
